# RustBoy

RustBoy is a GameBoy emulator written in Rust.

## Building the emulator

Install Rust (https://www.rust-lang.org/tools/install), then run the commands below:

```
git clone https://github.com/jpiechowka/rustboy.git
cd rustboy
cargo build --release
```
